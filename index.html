LearnQuest Study Prototype

Experimental interface for studying the effect of reward feedback on intrinsic motivation.
Select a condition below to preview.

A
Condition A

Reward Feedback

Gamified interface with XP points and completion-based badges after each task.

+50 XP awarded per task (completion-based)
3 milestone badges (completion-based)
Animated reward feedback
Identical welcome & completion screens
B
Condition B

No Feedback

Identical tasks and interface without any reward elements or gamification.

Same learning tasks
Same visual design
Neutral completion messages
No reward elements
Key Design Principle: Both conditions use the exact same learning tasks, layout, typography, and colour scheme. The only difference is the presence of reward feedback (XP points and badges) in Condition A.
